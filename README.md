# Basic Todo Application API in Flask

### Python Version - **3.10**

## Step to run the project

-   Clone the project
    ```
    git clone https://github.com/mmw-ani/todo-flask
    cd todo-flask
    ```
-   Now start the virtual environment
    ```
    source venv/bin/activate
    ```
-   Install all the required packages
    ```
    pip install -r requirements.txt
    ```
-   Now finally start the server
    ```
    	python main.py
    ```
